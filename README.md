## <img src="https://raw.githubusercontent.com/rosgear/rg-be-references-folder_profiles/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Справочник «Профили медиапапок»

Справочник профилей настроек отображения медиапапок.

Справочник входит в состав Панели управления GePanel и cлужит расширением для [модуля «Справочники»](https://github.com/rosgear/rg-be-reference).

### Скриншот
<img src="https://github.com/rosgear/rg-be-references-folder_profiles/blob/main/assets/help/grid.png?raw=true">

### Установка

Для добавления справочника в ваш проект, вы можете просто выполнить команду ниже:

```
$ composer require rosgear/rg-be-references-folder_profiles
```

или добавить в файл composer.json вашего проекта:
```
"require": {
    "rosgear/rg-be-references-folder_profiles": "*"
}
```
или скачать архив на [странице справочника](https://rosgear.ru/component/rg-be-references-folder_profiles/) в каталоге приложений RosGear.

После добавления справочника в проект выполните его установку в редакцию веб‑приложения с помощью Панели управления GePanel.

Reference of media folder display settings profiles.

The reference is part of the GePanel Control Panel and serves as an extension for the [«References» module](https://github.com/rosgear/rg-be-reference).

### Screenshot
<img src="https://github.com/rosgear/rg-be-references-folder_profiles/blob/main/assets/help/grid.png?raw=true">

### Installation

To add the reference to your project, you can simply run the command below:

```
$ composer require rosgear/rg-be-references-folder_profiles
```

or add to your project's composer.json file:
```
"require": {
    "rosgear/rg-be-references-folder_profiles": "*"
}
```
or download the archive from the [reference page](https://rosgear.ru/component/rg-be-references-folder_profiles/) in the RosGear application catalog.

After adding the reference to the project, install it into the web application edition using the GePanel Control Panel.
